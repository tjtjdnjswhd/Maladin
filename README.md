# Maladin API
ASP.NET Core, SQL Server로 구현한 온라인 서점 API 프로젝트

## 기능 목록
### 계정
- [ ] 가입
  - [ ] ID/Password 가입
  - [ ] OAuth 가입
- [ ] 탈퇴
- [ ] 업데이트
  - [ ] 이름
  - [ ] 비밀번호
  - [ ] 이메일
- [ ] 이메일 인증
- [ ] OAuth 연동/해제
- [ ] 마일리지

### 쇼핑
- [ ] 검색
- [ ] 장바구니
- [ ] 보관함
- [ ] 구매목록
- [ ] 기본 배송지
- [ ] 결제 시스템

## 기술 스택
 * ASP.NET Core 7.0
 * SQL Server
 * EF Core
 * <a href="https://www.iamport.kr/">iamport</a>
